# Shokunin PRD Review

## Overview

Use this skill to review PRDs and product proposals.

## Checks

- Problem clarity
- Evidence
- Target segment
- MVP scope
- Primary metric
- Guardrails
- Requirements
- Technical feasibility
- Simpler alternatives
- Cost / ROI
- Decision ask

## Red Flags

- "AI because AI is a trend"
- "Useful for all users"
- Metrics listed without baseline or target
- Vague requirements
- No experiment plan
- No score caps / guardrails
