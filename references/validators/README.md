# Validators

Validators are review lenses used by Shokunin Review.

## Core Validators

- **Evidence Reviewer** - Validates evidence quality and completeness
- **Metric Reviewer** - Validates metric definitions and tracking
- **Decision Reviewer** - Validates decision clarity and actionability  
- **Requirement Reviewer** - Validates requirement completeness
- **Technical Feasibility Reviewer** - Validates technical implementation feasibility
- **AI Guardrail Reviewer** - Validates AI safety and guardrails
- **Cost / ROI Reviewer** - Validates cost analysis and ROI calculations
- **Simplicity Reviewer** - Validates solution simplicity and avoids overengineering
- **Output Guard** - Validates output quality and completeness
