# Checkout A/B Test Plan (Before)

## Hypothesis

Adding a progress bar to checkout will increase completion rate.

## Background

Users are abandoning checkout because they don't know how long it will take.

## Design

We'll show a progress bar at the top of checkout showing 4 steps:
1. Cart
2. Shipping
3. Payment
4. Confirmation

## Metrics

- Completion rate
- Time to complete
- User satisfaction

## Sample Size

We'll run it for 2 weeks and see what happens.

## Analysis

We'll compare the completion rates between control and treatment groups.
