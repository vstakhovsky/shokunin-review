# Shokunin Review RFC

Review Technical Design documents.

## Usage

```bash
shokunin-review-rfc <file>
```

## What It Checks

- Technical decision clarity
- Alternatives analysis
- Trade-offs acknowledgment
- System boundaries
- API/data contracts
- Failure modes
- Rollout plan
- Test plan

## Score Caps

- No technical decision → max 60
- No alternatives → max 65
- No trade-offs → max 70
- No failure modes → max 70
