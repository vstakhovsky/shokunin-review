# Shokunin Improve

Suggest specific improvements for documents.

## Usage

```bash
shokunin improve <file> [--focus <area>]
```

## Focus Areas

```bash
--focus structure     # Document structure
--focus evidence      # Evidence and data
--focus metrics       # Metrics and measurement
--focus requirements  # Requirements clarity
--focus technical     # Technical approach
--focus strategy      # Strategic clarity
```

## What It Provides

- Prioritized fix suggestions
- Specific text examples
- Acceptance criteria
- Effort estimates
- Expected score improvement

## Example

```bash
shokunin improve docs/prd.md --focus metrics
```
