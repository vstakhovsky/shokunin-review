# Shokunin Review Strategy

Review Product Strategy documents.

## Usage

```bash
shokunin-review-strategy <file>
```

## What It Checks

- Strategic thesis clarity
- Target customer/ICP
- Evidence quality
- Opportunity sizing
- Trade-offs acknowledgment
- Sequencing logic
- Success metrics

## Score Caps

- No strategic thesis → max 60
- No target segment → max 60
- No opportunity sizing → max 65
- No trade-offs → max 70
- Generic strategy only → max 55
